# AI Rules - always follow

## Status

- **STATUS.md file:** current project status
- **read it:** after session starts
- **Update it:** after changes (Objective, Progress [x], Context, Next).

## General

**Be concise, but clear and precise**

- **Language:** see Language policy in `PROJECT.md`. Chat = Czech; `docs/` = Czech; control/meta files = English.

## Response

- **informal:** no fluff, no apologies, no praise.
- **Ack:** Single-line confirmation only.
